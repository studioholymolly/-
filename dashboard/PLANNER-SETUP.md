# 촬영 기획안 도우미 (/planner)

기획안을 써본 적 없는 브랜드가 **5단계 선택만으로 촬영 기획안(PPT·PDF)을 완성**하는 공개 페이지입니다.

- 주소: `https://holymolly-ops.vercel.app/planner` (임베드: `/planner?embed=1`)
- 로그인 불필요 — `/inquiry`와 같은 격리 구조

## 흐름 (v2 — 실무형 시안보드 구조)

실제 촬영 기획안(AIXNINE·THETIS 등) 구조 기반: 포토그래퍼·프롭 스타일리스트가 보고 **재료·소품·배경지를 준비**하고, 촬영에서 어떤 이미지를 찍어야 하는지 시각적으로 보는 문서.

1. **브랜드·제품** — 브랜드명 · 업종 · 제품 · **실물 사진 업로드(REAL IMAGE, 최대 4장 — 여러 제품 동시 촬영 대응)** · 샘플 수량
   - **브랜드 소개서 업로드(선택)**: Vercel 환경변수 `ANTHROPIC_API_KEY`가 설정된 경우에만 노출.
     PDF·이미지(3MB 이하)를 올리면 `/api/brand-mood`(Claude API)가 브랜드 무드 요약 + 키워드를 뽑아
     TONE & MOOD 페이지와 표지 키워드에 자동 반영. 분석 결과는 업로드 직후 직접 수정 가능.
2. **촬영 목적** — 촬영 유형(문의 폼과 동일 선택지) + 사용처 (복수)
3. **무드·배경지** — 무드 카드 최대 2개(팔레트가 배경지 색 제안으로 연결) + **무드 레퍼런스 이미지 업로드(최대 4장)**
4. **컷 시트** — 상세페이지 구조 순서(누끼→메인 KV→연출→제형·질감→디테일→사용컷)로 자동 제안, 컷마다:
   - **배경지 색** (프리셋 + 무드 톤 + 커스텀 색상, 누끼는 "배경 X")
   - **소품·원물** 텍스트
   - **레퍼런스 이미지 업로드** (컷당 1장)
   - 역할 배지 (메인 KV / 필수 / 얼터컷) · 사용처 · 포토그래퍼 메모
5. **일정·예산** — 날짜 · 예산 · 모델 · 샘플 전달 방법
6. **완성** — 표지 / PRODUCT / TONE&MOOD / **SHOT LIST**(전체 컷 체크리스트 표 — 현장에서 ✓ 체크) / **CUT 상세(컷당 1페이지**: 좌 제품·배경지, 중앙 레퍼런스 크게, 우 촬영 포인트·필요 소품 리스트, 영상은 SCENE 표기) / **PREP 준비물 체크리스트**(배경지별 컷 번호·소품 총괄·브랜드 전달 목록) / 진행 안내
   - 📄 **PDF 저장**: 인쇄 창에서 "PDF로 저장" (가로 A4)
   - 📊 **PPT 다운로드**: 업로드한 이미지 포함 .pptx — PowerPoint·구글 슬라이드에서 수정 가능
   - ✏️ **이 기획안 그대로 문의하기**: 컷별 배경지·소품 정보가 텍스트로 `/inquiry` 폼에 자동 입력 (이미지는 용량 문제로 제외 — 문의 첨부파일로 안내)

## /inquiry 연동 지점

- 랜딩 두 번째 버튼 "🐥 촬영 기획안 도우미" (코드 고정 — inquiry_site 데이터와 무관)
- 폼 3단계(기획안)에 "3분 기획안 도우미 써보기" 카드
- 도우미에서 넘어오면: 브랜드명·촬영 유형·목적·컨셉·기획안 4항목·일정·예산이 자동 입력되고
  `planStatus='가이드 작성'`이라 **대시보드 문의 상세에서 기존 UI 그대로** 보입니다. (`data.planner: true` 표시)

## 선택: 기획안 리드 저장

문의 없이 기획안만 만들고 떠난 브랜드도 기록하고 싶다면 Supabase SQL Editor에서
`supabase/planner-schema.sql`을 실행하세요 (`planner_plans` 테이블 — 익명 insert 전용).
**실행하지 않아도 도우미는 정상 동작합니다** (저장 실패는 조용히 무시).
